# SecureGen AI

SecureGen AI is a university Deep Learning and Cybersecurity project focused on detecting insecure patterns in AI-generated code. This repository contains the lab deliverables for project planning, exploratory data analysis, model benchmarking, and the product demo experience.

## Contents

- `lab1_project_plan.md`: Lab 1 project plan
- `lab1_architecture_diagram.md`: Lab 1 architecture diagram
- `lab2_eda_notebook.py`: Lab 2 EDA notebook-style script
- `lab2_eda_report.md`: Lab 2 EDA report
- `lab3_benchmark_notebook.py`: Lab 3 benchmarking notebook-style script
- `lab3_benchmark_report.md`: Lab 3 benchmarking report
- `lab4_landing_page.html`: Lab 4 landing page
- `lab4_demo_app.jsx`: Lab 4 React demo component
- `lab4_product_documentation.md`: Lab 4 user-facing documentation
- `index.html`: Runnable integrated single-file demo entry point

## Running the Demo

Open `index.html` in a browser, or serve the folder locally with a simple static server.
